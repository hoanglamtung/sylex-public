# Sylex Assistant Pro – Hackathon Code

This repository contains the client and server code for Sylex Assistant Pro, submitted to the Google for Startups AI Challenge.

## How to Run the Backend
1. cd server
2. npm install
3. Create a `.env` file with your own API keys:
   GOOGLE_API_KEY=YOUR_KEY
   FIREBASE_PROJECT_ID=YOUR_PROJECT
4. npm start

## How to Run the Mobile App
1. cd client-rn
2. npm install
3. For iOS:
   npx pod-install
   npx react-native run-ios
4. For Android:
   npx react-native run-android

## Notes
- All sensitive keys have been removed.
- Replace placeholders with your own keys to run the app.
- The code structure, agent logic, and architecture are fully included for judging.
